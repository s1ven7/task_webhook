# Webhook Service

## 项目简介

Webhook Service 是一个用于管理 Webhook 的 web 服务。用户可以创建、读取、更新和删除 Webhook，支持通过模板替换 URL 中的动态数据。

## 功能特性

- 支持 Webhook 的 CRUD 操作
- 触发 Webhook 时支持动态数据替换
- 使用 JSON 进行数据交互

## 技术栈

- **前端**：使用 Ant Design 和 React 构建用户界面
- **后端**：基于 Node.js 和 Express 实现 RESTful API

## 安装与使用

- **前端**：
    cd frontend
    npm install
    npm start

- **后端**:
    cd backend
    npm install
    npm start
